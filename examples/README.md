# Job Prep examples

4 个 skill，与 `skill-forge/built/job-prep/<skill>` 同构。

```bash
python skill-core/skill.py install examples/job-prep/job-prep-assistant --host cursor --scope global
```

- `job-prep-assistant` — 主入口（推荐）
- `resume-tailor` — 简历 tailoring
- `behavioral-interview` — STAR 行为面试
- `tech-interview-lite` — 技术面 lite

重新导出：`python skill-core/scripts/export_examples.py`
