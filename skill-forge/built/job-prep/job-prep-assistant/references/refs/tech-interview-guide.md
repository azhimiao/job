# 技术面试 lite 范围

## 覆盖（按级别裁剪）

| 级别 | 算法 | 基础 | 系统设计 |
|------|------|------|----------|
| 实习/校招 | 数组/链表/栈队列/二叉树 | OS/网络/DB 概念 | 不要求 |
| 1-3 年 | + 排序/二分/哈希/简单 DP | SQL 索引、HTTP、并发基础 | 单服务 QPS 估算 |
| 后端 Java | JVM 基础、Spring、MySQL、Redis | 微服务概念 | 缓存/消息队列场景 |

## 模拟题输出

`job/mock-questions.md` — 分类清单 + 参考思路（非标准答案抄袭）

## 练习方式

1. author 口述思路 → 反馈
2. 限时 15 min 小算法 → review 复杂度
3. 不写完整作弊代码库；引导理解

## 禁止

- 保证押中原题
- 代做在线测评（OA）作弊
