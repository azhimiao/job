# JD 解析指南

## 提取字段

1. **硬技能** — 语言、框架、工具（must-have vs nice-to-have）
2. **职责** — 动词 + 对象（负责/参与/优化…）
3. **级别信号** — 实习 / 校招 / 3年+ / 专家
4. **文化信号** — 快节奏、国企稳定、产品导向

## 输出 mapping

`job/jd-analysis.yaml`:

```yaml
must_have: []
nice_to_have: []
responsibilities: []
keywords_for_resume: []
red_flags: []  # 不合理要求、过度加班暗示
```

## 简历对齐

- 每条 bullet：动词 + 技术/方法 + 结果
- 仅使用 author **真实**经历；gap 用「可迁移技能」表述，不虚构

## 社会学参考

劳动力市场 **信号理论**（Spence）：学历与项目经历是能力信号，虚假信号损害长期声誉。
